# discord-ip-logger
A Discord IP Logger that can be used for anything your mind wants to do with those poor IP's


1. How to use this IP Logger

First you need to go to your discord server's settings and create a webhook, then copy the webhook url and paste it where it says "YOUR WEBHOOK HERE".
Now you can upload it to the internet using a tool like https://pythonanywhere.com

You are done, enjoy.
